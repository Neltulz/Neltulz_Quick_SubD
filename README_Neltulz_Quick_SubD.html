Key Features

  • Quickly change subdivision modes between OFF, ON, and ON+ (Cage).
  • Support for both Relative and Specific subdivision level settings.
  • Always Pick Best Shading so that no matter what mode you're viewing, your object will have the best normal shading.
  • Easily toggle on/off wireframes and subdivision wireframes
  • Easily toggle on/off distracting edge colors for visualization purposes.
  • Permanently apply subdivision modifier to multiple objects simultaneously.
  • Subdivision warning popups to help prevent program instabilty and crashing from choosing too high of a subdivision level.

Installation

  1. Download the Neltulz - Quick SubD zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Quick SubD zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Generic: Neltulz - Quick SubD, enable the checkbox.
  8. At the lower left of the preferences window, click Save Preferences and close the preferences window.
